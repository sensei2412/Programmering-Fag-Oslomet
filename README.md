# CodeVault – Programmeringsprosjekter fra OsloMet

CodeVault er en samling av programmeringsoppgaver, øvinger og prosjekter utviklet gjennom ulike programmeringsfag ved OsloMet.

Repositoryet inneholder arbeid relatert til algoritmer, datastrukturer, objektorientert programmering og problemløsning i Java. Prosjektene fokuserer på å utvikle forståelse for grunnleggende og avanserte konsepter innen programvareutvikling.

## Innhold

Repositoryet inneholder blant annet:
- Java-oppgaver
- Algoritmer og datastrukturer
- Objektorientert programmering
- Sorteringsalgoritmer
- Rekursjon
- Arrays og lister
- Filhåndtering
- Logiske problemløsningsoppgaver
- Praktiske øvinger fra OsloMet

## Teknologier brukt

- Java
- IntelliJ IDEA
- Git / GitHub
- Maven

## Temaer i prosjektet

Prosjektene dekker flere sentrale områder innen programmering:

### Objektorientert programmering
- Klasser og objekter
- Arv og polymorfisme
- Innkapsling
- Metoder og konstruktører

### Algoritmer og datastrukturer
- Sortering
- Søkealgoritmer
- Lister og arrays
- Binære trær
- Rekursjon

### Problemløsning
- Effektive løsninger
- Analyse av algoritmer
- Optimalisering
- Strukturert kode
